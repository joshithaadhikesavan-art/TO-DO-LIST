#  Animated To-Do List App

A simple and interactive **To-Do List web app** built using HTML, CSS, and JavaScript with modern UI and animations.

##  Features

* Add, edit, and delete tasks
* Mark tasks as completed 
* Filter (All / Completed / Pending)
* Data saved using Local Storage 
* Animated background & smooth UI 

## Tech Stack

* HTML
* CSS
* JavaScript
